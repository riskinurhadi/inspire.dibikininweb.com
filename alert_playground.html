DibikininAlerts.

Alert Cantik, Tanpa Ribet.

Library notifikasi Javascript ringan (2KB) dengan desain modern meniru SweetAlert2. Tanpa dependensi, tanpa CSS eksternal.

Informasi umum sistem.

Cara Penggunaan

Cukup panggil fungsi Dibikinin.fire() di dalam script kamu. Library ini sudah menangani Promise, jadi kamu bisa menggunakan .then() atau await.

1. Basic Usage
// Panggil alert sederhana
Dibikinin.fire({
    title: 'Halo!',
    message: 'Ini adalah pesan alert sederhana.',
    type: 'info'
});
2. Async/Await (Menunggu Tombol ditekan)
async function hapusData() {
    // Alert akan muncul dan script akan 'pause' di sini
    await Dibikinin.fire({
        title: 'Hapus Data?',
        message: 'Data tidak bisa dikembalikan setelah dihapus.',
        type: 'warning',
        buttonText: 'Ya, Hapus'
    });

    // Kode di bawah ini hanya jalan SETELAH user klik tombol
    console.log('Data berhasil dihapus!');
}
